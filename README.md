This project demonstrates the deployment and operation of a small-scale
Security Operations Center (SOC) using Wazuh.

## Objectives
- Deploy a SIEM in a Linux server environment
- Collect and analyze endpoint logs
- Detect and investigate real attack activity
- Document incidents using SOC-style reporting

## Environment
- Hypervisor: VirtualBox
- SIEM: Wazuh 4.7
- Server OS: Ubuntu Server
- Endpoints: Windows 10
- Attacker: Kali Linux

## Project Phases
1. Infrastructure & networking
2. Wazuh deployment
3. Endpoint onboarding
4. Attack simulation
5. Alert investigation & reporting

See `/docs` for detailed setup and troubleshooting notes.
